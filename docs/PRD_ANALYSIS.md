# PRD 需求分析文档

> 基于《视频转AI笔记工具 产品需求文档 V1.0》（2026-06-05）

## 1. 产品定位

| 维度 | 说明 |
|------|------|
| 定位 | 面向学习场景的轻量化 AI 工具 |
| 用户 | 技术学习者、网课学习者 |
| 部署 | 初期本地部署，后续 Docker |
| 开源 | 个人自用 → 身边学习者 → 最终开源 |

## 2. 核心业务流程

```
用户输入(本地视频/链接+Cookie)
  → [B站] 优先拉官方字幕（player API）
      ├─ 有字幕 → 跳过 ASR
      └─ 无字幕 → 下载视频 → 抽音频 → ASR 转写（引擎可配置）
  → 清洗逐字稿
→ AI结构化笔记(双风格) → 术语+延伸推荐 → 用户编辑
→ 导出/飞书同步/问答/分享 → 历史任务
```

## 3. 模块需求拆解与验收标准

### 3.1 视频输入模块

| 需求 ID | 描述 | 验收标准 |
|---------|------|----------|
| IN-01 | 本地上传 mp4/mov | 上传成功返回文件路径，可创建任务 |
| IN-02 | B站链接 + Cookie | Cookie 用于 yt-dlp 下载视频、B 站 player API 拉字幕（AI 字幕需 SESSDATA）、延伸视频搜索；权限与登录账号一致 |
| IN-03 | 抖音公开视频 | 不支持合集/分集链接，失败有明确提示 |
| IN-04 | Cookie 失效提示 | 解析失败时返回可读错误，不自动重试 |

### 3.2 任务管理模块

| 需求 ID | 描述 | 验收标准 |
|---------|------|----------|
| TK-01 | 异步处理 | 提交后可离开，后台继续执行 |
| TK-02 | 三态列表 | 处理中 / 失败 / 已完成 可筛选 |
| TK-03 | 失败原因 | 展示具体原因（解析/转写/LLM 等） |
| TK-04 | 历史记录 | 服务端持久化全部任务 |

### 3.3 逐字稿模块

| 需求 ID | 描述 | 验收标准 |
|---------|------|----------|
| TR-01 | 带时间戳全量逐字稿 | 独立 Tab 展示 segments |
| TR-02 | 过滤语气词 | 嗯、啊、那个、然后等被清理 |
| TR-03 | 剔除复读 | 连续重复无意义语句去重 |
| TR-04 | 多语言 | 非中文保留原文，笔记翻译中文 |
| TR-05 | 时间戳跳转 | 点击 segment 定位视频播放位置 |
| TR-06 | B 站字幕优先 | B 站链接先尝试官方字幕；成功则跳过 ASR，逐字稿 `source` 为 bilibili_subtitle |
| TR-07 | 转写引擎可切换 | 设置页可选必剪/快手/Whisper 本地/Groq；无字幕或本地/抖音视频走所选引擎 |

### 3.4 结构化笔记模块

| 需求 ID | 描述 | 验收标准 |
|---------|------|----------|
| NT-01 | 视频基础信息 | 标题、来源、生成时间 |
| NT-02 | 概述总结 | 约 100 字总览 |
| NT-03 | 视频目录 | 章节导航锚点 |
| NT-04 | 分章节内容 | 每章含 AI 重点 + 原文实录 |
| NT-05 | 术语解释 | 自动识别；小白版含比喻 |
| NT-06 | 延伸知识点 | 关键词列表供 B 站搜索 |
| NT-07 | 双风格 | 小白学习版 / 专业精简版 |
| NT-08 | 详略模式 | 详细模式 / 精简模式 |

### 3.5 延伸推荐模块

| 需求 ID | 描述 | 验收标准 |
|---------|------|----------|
| RC-01 | B站搜索 | 每知识点 1-2 个相关视频 |
| RC-02 | 关键词编辑 | 用户可修改并刷新列表 |
| RC-03 | 仅展示 | 不自动触发笔记生成 |

### 3.6 AI 问答模块

| 需求 ID | 描述 | 验收标准 |
|---------|------|----------|
| CH-01 | 独立 Tab | 笔记详情页「AI 问答答疑」 |
| CH-02 | 上下文 | 默认基于逐字稿 + 笔记 |
| CH-03 | 自由提问 | 支持脱离笔记提问 |
| CH-04 | 历史持久化 | 关闭重开可继续对话 |
| CH-05 | 风格同步 | 话术匹配笔记生成风格 |

### 3.7 飞书同步模块

| 需求 ID | 描述 | 验收标准 |
|---------|------|----------|
| FS-01 | OAuth 授权 | 用户提前授权飞书账号 |
| FS-02 | 手动同步 | 点击按钮触发，非自动 |
| FS-03 | 文件夹选择 | 授权后可选择存储目录 |
| FS-04 | 云文档 | 同步完整笔记，保留层级 |
| FS-05 | 多维表格 | 新增记录，链接/本地两套字段 |

### 3.8 导出模块

| 需求 ID | 描述 | 验收标准 |
|---------|------|----------|
| EX-01 | Markdown 导出 | 导出编辑后最终版 |
| EX-02 | PDF 导出 | 保留层级与高亮颜色 |

### 3.9 分享模块

| 需求 ID | 描述 | 验收标准 |
|---------|------|----------|
| SH-01 | 公开链接 | UUID token，无需登录 |
| SH-02 | 只读全板块 | 目录/总结/逐字稿/推荐均可见 |

## 4. 非功能性需求

| 类别 | 要求 |
|------|------|
| 部署 | 本地 + Docker |
| 存储 | 原视频处理后保留本地 |
| 隐私 | 分享后公开可访问 |
| 异常 | 明确错误提示，不自动重试 |

## 5. 版本规划对照

### MVP 一期（必做）
- 全部 3.1–3.9 模块核心能力

### 二期迭代
- 大文件分片上传、合集分类、自定义风格模板、更多平台、自定义高亮颜色

### 三期选做
- 思维导图、PPT 生成、问答插入笔记

## 6. 技术决策（默认方案）

| 待确认项 | 默认方案 |
|----------|----------|
| B站搜索 | 公开搜索 API + Cookie，失败降级为仅关键词 |
| B 站逐字稿来源 | 官方字幕优先（player API），失败降级 ASR |
| ASR 引擎 | 默认必剪；可选快手、fast-whisper 本地、Groq API；网页设置页切换 |
| 术语释义 | LLM 为主，可选搜索引擎兜底 |
| 分享权限 | UUID 公开链接 + 快照 |
| 笔记编辑 | Markdown 双栏，`==高亮==` 语法 |
| 飞书 | 企业自建应用 OAuth |

## 7. 外部依赖清单

- FFmpeg（系统安装）
- 飞书开放平台 App ID / Secret
- LLM API Key（用户 UI 配置）
- ASR 引擎：必剪（默认）/ 快手 / fast-whisper 本地 / Groq API；`GROQ_API_KEY` 仅 Groq 需要
